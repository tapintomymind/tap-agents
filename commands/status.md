# /status

Ask EA for a current briefing across all projects.

## Usage

```
/status
```

## What You Get

Standardized executive briefing (~250-400 words) with:
- ACTIVE PROJECTS (per project: phase, last activity, awaiting, blockers)
- PAUSED projects
- DECISIONS NEEDED (queue)
- TEAM HEALTH (Org Designer signals)
- Optionally: RECENT FYI

## See Also

- `/briefing` — same as `/status` but allows EA to add more context (RECENT FYI, scheduled-cadence emphasis)
- `/queue` — just the decisions queue, nothing else
- `/inbox` — flagged items + FYI feed
