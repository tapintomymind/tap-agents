# /intake

Engage Intake directly. Use this if you want to start interviewing without a specific idea pre-stated, or to re-intake a paused/shifted project.

## Usage

```
/intake [optional: project context]
```

## What Happens

Intake activates and starts asking hard-hitting questions about whatever's on your mind. If you provide context, Intake uses it as the seed. If not, Intake asks "what's the work?"

## When to Use

- You have a vague idea and want help shaping it
- You want to re-interview a paused project before resuming
- You want to update a brief mid-project (scope shift)
- An agent returned WRONG_AGENT pointing to a requirements gap

## See Also

- `/team` — same thing but with explicit seed in the command
- `/feature` — Intake in **feature mode** for ideating a feature inside an existing project (anchors to live PRD/scope, produces a feature-brief)
- `/strategist` — skip intake (only if brief already exists and approved)
- `/status` — see paused projects you might want to re-intake
